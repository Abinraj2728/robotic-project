📘 Project Overview

💡 Automatic Light Control

Motion sensor (PIR) detects presence and controls lighting with 5-second timeout after last motion.

Sensor: PIR HC-SR501

🌡️ Automatic Fan Control

Temperature sensor (DHT22) monitors room temperature. Fan activates above 30°C.

Sensor: DHT22

📱 Remote Control Simulation

Serial commands simulate mobile app control. Type '1', '0', or 'A' in Serial Monitor.

Interface: Serial Commands

🎮 Live Simulation

📱 Quick Serial Commands: In the simulation, click the Serial Monitor button (🔌) and type: '1' - Lamp ON | '0' - Lamp OFF | 'A' - AUTO Mode

💡 How It Works

🔧 Components

• Arduino (simulated)
• DHT22 Temperature Sensor
• PIR Motion Sensor
• 2x Relay Modules

⚙️ Automation Logic

• Fan: ON when temp ≥ 30°C
• Light: ON with motion, OFF after 5s
• Manual override via serial

🔌 Pin Connections

• DHT22 → Pin 2
• PIR → Pin 3
• Lamp Relay → Pin 4
• Fan Relay → Pin 5

📋 Command Reference

Command Function Mode
'1' Turn Lamp ON Manual Override
'0' Turn Lamp OFF Manual Override
'A' Return to AUTO mode Automatic
