🌓 Task 2: Smart Circadian Lighting System

Variable Control System - Color temperature changes throughout the day

🎨 Color Cycle (60-second simulated day): Sunrise → Morning → Noon → Afternoon → Sunset → Evening → Night
🌓
Smart Circadian Lighting System
Task 2: RGB LED with color temperature control
In Progress

🔧 Components

  • • Arduino Uno
  • • RGB LED (Common Cathode)
  • • LDR (Photoresistor)
  • • 3x 220Ω Resistors

🎨 Color Temperatures

  • • 🌅 Sunrise: Warm orange
  • • ☀️ Noon: Bright daylight
  • • 🌇 Sunset: Deep orange
  • • 🌙 Night: Dim red

⚙️ Pin Connections

  • • Red LED → Pin 9 (PWM)
  • • Green LED → Pin 10 (PWM)
  • • Blue LED → Pin 11 (PWM)
  • • LDR → A0

📋 How It Works

  • • Simulates a full day in 60 seconds
  • • Smooth transitions between colors
  • • LDR detects ambient light for compensation
  • • Supports natural sleep-wake cycles
